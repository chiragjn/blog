The Coming Soon Plugin for Pelican

Want to keep your pelican blog readers excited with what you are working with? I made a small plugin for Pelican which allows you to add “Coming Soon” Status to your blog posts.Your readers will see these articles listed on your blog but not its content.

The plugin is open source and available on Github . Please see the README.md on on Github on how to integrate and use this plugin.

P.S. This plugin can make you lazy! P.S.S. This plugin is not completely tested and runs a delete over output drafts to remove them from published
